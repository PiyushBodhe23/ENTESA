About Embedded Club

The Embedded Club focuses on microcontrollers, firmware development, and real-time systems. It emphasizes low-level programming and direct interaction with hardware.

Core Areas

Microcontrollers

Work with PIC, STM32, and AVR architectures.

Firmware Development

Write efficient C code to control hardware directly.

Real-Time Systems

Understand timing, interrupts, and RTOS concepts.

Skills You Gain

Embedded C STM32 / PIC Interrupts Timers UART / SPI / I2C

Why Embedded?

Embedded systems power almost every electronic device. This club builds the foundation for working at the hardware level.
